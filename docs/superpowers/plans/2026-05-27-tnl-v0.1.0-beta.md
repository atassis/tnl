# tnl v0.1.0-beta Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship v0.1.0-beta from current v0.1.0-alpha state: installers + pairing + inspector + status/stop + admin CLI + reconnect + healthcheck + random subdomain + full CLI hygiene pass + OSS publication infrastructure.

**Architecture:** No new runtime processes — `tnld` and `tnl` binaries grow internally via new subcommands and modules. Pairing replaces the current "copy plaintext token from server file" handoff with a 5-min one-time code redeemed via REST. Reattach extends the existing yamux session lifecycle with a grace window. CI/release/Docker workflows live in `.github/`. Installer scripts in `scripts/` are POSIX sh.

**Tech Stack:** Rust 1.94 workspace; existing crates (`tnl-protocol`, `tnld`, `tnl`, `tnl-e2e`); new deps `dialoguer`, `clap_complete`, `clap_mangen`, `built`, `governor`, `expectrl` (dev-only). Installers: POSIX sh + bats tests. CI: GitHub Actions + cross-rs for musl. Docker: existing multi-stage Dockerfile, multi-arch via buildx.

**Spec:** [`docs/superpowers/specs/2026-05-27-tnl-v0.1.0-beta-design.md`](../specs/2026-05-27-tnl-v0.1.0-beta-design.md). When this plan and the spec disagree, code wins; if neither is clear, file an issue and stop.

**Conventions (apply to every task):**
- Conventional Commits: `feat(scope):`, `fix(scope):`, `test(scope):`, `chore:`, `docs:`, `refactor:`. Scopes: `protocol`, `tnld`, `tnl`, `e2e`, `deploy`, `ci`, `scripts`.
- Before each commit: `cargo fmt --all`, `cargo clippy --workspace --all-targets -- -D warnings`, `cargo test --workspace` (or scoped test if iterating).
- `unsafe_code = "forbid"` workspace-wide. Don't add `unsafe`.
- One task = one commit by default. If a task naturally splits into two commits (e.g., wire types + handler), say so explicitly in the task.
- Never skip pre-commit hooks (`--no-verify`).

```rust
//! Regression test for the alpha-deploy bug:
//!     async-tungstenite was compiled without a TLS feature, so
//!     `connect_async("wss://...")` failed with
//!     `URL error: TLS support not compiled in`.
//!
//! This test stands up a minimal rustls-backed TCP listener that accepts
//! a TLS handshake (cert is self-signed via rcgen), then verifies that
//! `tnl::client::connect_and_create`-style dial of `wss://` makes it past
//! the TLS layer. We assert specifically that the failure is NOT the
//! "TLS support not compiled in" symptom.

use std::net::SocketAddr;
use std::sync::Arc;
use std::time::Duration;

use rustls::pki_types::{CertificateDer, PrivateKeyDer};
use tokio::io::{AsyncReadExt, AsyncWriteExt};
use tokio::net::TcpListener;

#[tokio::test(flavor = "multi_thread", worker_threads = 2)]
async fn wss_dial_does_not_report_tls_feature_missing() {
    // 1) Generate a self-signed cert for "localhost".
    let cert = rcgen::generate_simple_self_signed(vec!["localhost".into()]).unwrap();
    let cert_der = CertificateDer::from(cert.cert.der().to_vec());
    let key_der = PrivateKeyDer::try_from(cert.signing_key.serialize_der()).unwrap();

    // 2) Bind a TCP listener; spawn a task that accepts ONE TLS handshake then drops.
    let listener = TcpListener::bind("127.0.0.1:0").await.unwrap();
    let addr: SocketAddr = listener.local_addr().unwrap();

    let server_config = rustls::ServerConfig::builder()
        .with_no_client_auth()
        .with_single_cert(vec![cert_der.clone()], key_der)
        .unwrap();
    let acceptor = tokio_rustls::TlsAcceptor::from(Arc::new(server_config));

    tokio::spawn(async move {
        if let Ok((sock, _)) = listener.accept().await {
            let mut tls = match acceptor.accept(sock).await {
                Ok(t) => t,
                Err(_) => return,
            };
            // Read any bytes, drop the connection.
            let mut buf = [0u8; 256];
            let _ = tls.read(&mut buf).await;
            let _ = tls.shutdown().await;
        }
    });

    // 3) Build a client root store that trusts our self-signed cert.
    let mut roots = rustls::RootCertStore::empty();
    roots.add(cert_der).unwrap();
    let _client_cfg = rustls::ClientConfig::builder()
        .with_root_certificates(roots)
        .with_no_client_auth();
    // (We don't pass the cfg into connect_async — async-tungstenite uses its
    //  bundled webpki-roots. The cert won't validate; we accept ANY failure
    //  except the specific "TLS support not compiled in" one.)

    let url = format!("wss://localhost:{}/control", addr.port());
    let req = http::Request::builder()
        .method("GET")
        .uri(&url)
        .header("Host", format!("localhost:{}", addr.port()))
        .header("Upgrade", "websocket")
        .header("Connection", "Upgrade")
        .header("Sec-WebSocket-Key", tungstenite::handshake::client::generate_key())
        .header("Sec-WebSocket-Version", "13")
        .body(())
        .unwrap();

    let result = tokio::time::timeout(
        Duration::from_secs(5),
        async_tungstenite::tokio::connect_async(req),
    )
    .await
    .expect("dial should not hang");

    match result {
        Ok(_) => {
            // Unexpected success (we used webpki-roots which don't trust self-signed),
            // but it means TLS works — also fine for this regression test.
        }
        Err(e) => {
            let msg = format!("{e:#}").to_lowercase();
            assert!(
                !msg.contains("tls support not compiled in"),
                "TLS feature missing on async-tungstenite — regression: {e:#}"
            );
        }
    }
}
```

```rust
use crate::auth::TokensFile;

#[derive(Subcommand, Debug)]
pub enum TokenCmd {
    /// List all tokens (name + hash prefix, no plaintext).
    List {
        /// Path to tokens.toml. Defaults to /etc/tnld/tokens.toml.
        #[arg(long, default_value = "/etc/tnld/tokens.toml")]
        tokens_file: PathBuf,
        /// Print JSON instead of a table.
        #[arg(long)]

    },

struct TokenSummary<'a> {
    name: &'a str,
    hash_prefix: String,
}

pub fn run(cmd: TokenCmd) -> Result<()> {
    match cmd {
        TokenCmd::List { tokens_file, json } => {
            let raw = std::fs::read_to_string(&tokens_file)
                .with_context(|| format!("read tokens file at {}", tokens_file.display()))?;
            let tf: TokensFile = toml::from_str(&raw).context("parse tokens file")?;
            if json {
                let out: Vec<TokenSummary<'_>> = tf
                    .tokens
                    .iter()
                    .map(|t| TokenSummary {
                        name: &t.name,
                        hash_prefix: hash_prefix(&t.hash),
                    })
                    .collect();
                println!("{}", serde_json::to_string(&out)?);
            } else {
                println!("{:<20} {:<30}", "NAME", "HASH PREFIX");
                for t in &tf.tokens {
                    println!("{:<20} {}", t.name, hash_prefix(&t.hash));
                }
            }
            Ok(())
        }
    }
}
```

```rust
fn add(name: String, tokens_file: PathBuf, replace: bool) -> Result<()> {
    // Load existing.
    let mut tf: TokensFile = match std::fs::read_to_string(&tokens_file) {
        Ok(raw) if !raw.trim().is_empty() => toml::from_str(&raw).context("parse tokens file")?,
        _ => TokensFile { tokens: vec![] },
    };

    // Duplicate-name check.
    let existing_idx = tf.tokens.iter().position(|t| t.name == name);
    if existing_idx.is_some() && !replace {
        anyhow::bail!("token {name:?} already exists; use --replace or choose another name");
    }

    // Generate plaintext and hash.
    let plaintext = format!(

        // 26-char base32-ish slice of random bytes; readable and dense.
        random_token_suffix(26)
    );
    let hash = hash_plaintext(&plaintext).context("argon2 hash")?;
    let entry = TokenEntry { name: name.clone(), hash };

    if let Some(i) = existing_idx {
        tf.tokens[i] = entry;
    } else {
        tf.tokens.push(entry);
    }

    // Atomic rewrite.
    write_tokens_file_atomic(&tokens_file, &tf)?;

    eprintln!("✓ Token {:?} written to {}", name, tokens_file.display());
    println!("{}", plaintext);
    Ok(())

fn random_token_suffix(len: usize) -> String {
    use rand::distributions::Slice;
    use rand::Rng;
    const ALPHABET: &[u8] = b"ABCDEFGHJKMNPQRSTUVWXYZabcdefghijkmnpqrstuvwxyz23456789";
    let mut rng = rand::thread_rng();
    let dist = Slice::new(ALPHABET).expect("alphabet non-empty");
    (0..len).map(|_| *rng.sample(&dist) as char).collect()
}

fn write_tokens_file_atomic(path: &std::path::Path, tf: &TokensFile) -> Result<()> {
```

```rust
fn revoke(name: String, tokens_file: PathBuf, yes: bool) -> Result<()> {
    let raw = std::fs::read_to_string(&tokens_file)
        .with_context(|| format!("read tokens file at {}", tokens_file.display()))?;
```

```rust
use std::sync::RwLock;
use std::time::SystemTime;

pub struct TokenStore {
    path: PathBuf,
    inner: RwLock<TokenStoreInner>,
}

struct TokenStoreInner {
    tokens: Vec<TokenEntry>,
    mtime:  SystemTime,
}

impl TokenStore {
    pub fn load(path: PathBuf) -> Result<Self> {
        let (tokens, mtime) = read_with_mtime(&path)?;
        Ok(Self { path, inner: RwLock::new(TokenStoreInner { tokens, mtime }) })
    }

    fn reload_if_changed(&self) -> Result<()> {
        let current_mtime = std::fs::metadata(&self.path)?.modified()?;
        if current_mtime > self.inner.read().unwrap().mtime {
            let (tokens, mtime) = read_with_mtime(&self.path)?;
            let mut guard = self.inner.write().unwrap();
            *guard = TokenStoreInner { tokens, mtime };
        }
        Ok(())
    }

    pub fn verify(&self, plaintext: &str) -> Option<String /* name */> {
        let _ = self.reload_if_changed();
        let guard = self.inner.read().unwrap();
        for t in &guard.tokens {
            if argon2_verify(&t.hash, plaintext).is_ok() {
                return Some(t.name.clone());
            }
        }
        None
    }
}

fn read_with_mtime(path: &Path) -> Result<(Vec<TokenEntry>, SystemTime)> {
    let raw = std::fs::read_to_string(path)
        .with_context(|| format!("read tokens file at {}", path.display()))?;
    let tf: TokensFile = toml::from_str(&raw)?;
    let mtime = std::fs::metadata(path)?.modified()?;
```

```rust
//! In-memory pairing registry. 5-min TTL, IP rate limit, attempt counter,
//! 16 concurrently-live codes max, one-time-use semantics.

use std::net::IpAddr;
use std::num::NonZeroU32;
use std::sync::atomic::{AtomicU8, Ordering};
use std::sync::Arc;
use std::time::{Duration, Instant, SystemTime};

use anyhow::Result;
use dashmap::DashMap;
use governor::clock::DefaultClock;
use governor::state::keyed::DashMapStateStore;
use governor::{Quota, RateLimiter};

const ALPHABET: &[u8] = b"ABCDEFGHJKMNPQRSTUVWXYZ23456789";
const CODE_LEN: usize = 6;
pub const MAX_CODES: usize = 16;
pub const MAX_ATTEMPTS_PER_CODE: u8 = 5;

pub struct PairEntry {
    pub code:       String,
    pub name:       String,
    pub created_at: Instant,
    pub expires_at: Instant,
    pub attempts:   AtomicU8,
}

#[derive(Debug, thiserror::Error)]
pub enum RedeemErr {
    #[error("pair_not_found")]
    NotFound,
    #[error("pair_expired")]
    Expired,
    #[error("pair_too_many_attempts")]
    TooManyAttempts,
    #[error("rate_limited; retry after {retry_after_sec}s")]
    RateLimited { retry_after_sec: u32 },
}

pub struct PairRegistry {
    entries: DashMap<String, Arc<PairEntry>>,
    by_ip:   RateLimiter<IpAddr, DashMapStateStore<IpAddr>, DefaultClock>,
}

impl PairRegistry {
    pub fn new() -> Self {
        let quota = Quota::per_minute(NonZeroU32::new(3).unwrap());
        Self { entries: DashMap::new(), by_ip: RateLimiter::dashmap(quota) }
    }

    pub fn create(&self, name: String, ttl: Duration) -> Result<(String, SystemTime)> {
        if self.entries.len() >= MAX_CODES {
            anyhow::bail!("pair registry full (max {MAX_CODES} concurrent codes)");
        }
        let raw = random_code();
        let entry = Arc::new(PairEntry {
            code: raw.clone(),
            name,
            created_at: Instant::now(),
            expires_at: Instant::now() + ttl,
            attempts: AtomicU8::new(0),
        });
        self.entries.insert(raw.clone(), entry);
        Ok((format_code(&raw), SystemTime::now() + ttl))
    }

    pub fn redeem(&self, code: &str, from_ip: IpAddr) -> Result<String, RedeemErr> {
        if self.by_ip.check_key(&from_ip).is_err() {
            return Err(RedeemErr::RateLimited { retry_after_sec: 30 });
        }
        let normalised = normalise(code);
        let entry = self.entries.get(&normalised).ok_or(RedeemErr::NotFound)?;
        if Instant::now() > entry.expires_at {
            drop(entry); self.entries.remove(&normalised);
            return Err(RedeemErr::Expired);
        }
        let attempts = entry.attempts.fetch_add(1, Ordering::AcqRel);
        if attempts >= MAX_ATTEMPTS_PER_CODE {
            drop(entry); self.entries.remove(&normalised);
            return Err(RedeemErr::TooManyAttempts);
        }
        let name = entry.name.clone();
        drop(entry);
        self.entries.remove(&normalised);
        Ok(name)
    }

    pub fn cleanup(&self) {
        let now = Instant::now();
        self.entries.retain(|_, e| now <= e.expires_at);
    }

    pub fn list(&self) -> Vec<(String, String, SystemTime)> {
        let now_inst = Instant::now();
        let now_sys = SystemTime::now();
        self.entries.iter().map(|kv| {
            let e = kv.value();
            (format_code(&e.code), e.name.clone(),
             now_sys + e.expires_at.saturating_duration_since(now_inst))
        }).collect()
    }
}

fn random_code() -> String {
    use rand::Rng;
    let mut rng = rand::thread_rng();
    (0..CODE_LEN).map(|_| ALPHABET[rng.gen_range(0..ALPHABET.len())] as char).collect()
}

fn format_code(raw: &str) -> String {
    let mut out = String::with_capacity(raw.len() + 2);
    for (i, c) in raw.chars().enumerate() {
        if i > 0 && i % 2 == 0 { out.push('-'); }
        out.push(c);
    }
    out
}

pub fn normalise(s: &str) -> String {
    s.chars().filter(|c| c.is_ascii_alphanumeric()).map(|c| c.to_ascii_uppercase()).collect()
}
```

```rust
use std::net::SocketAddr;
use std::time::Duration;
use axum::extract::{ConnectInfo, Json, State};
use axum::http::{HeaderMap, StatusCode};
use axum::response::IntoResponse;
use axum::routing::{get, post};
use axum::Router;
use serde_json::json;
use tnl_protocol::messages::{PairCreateReq, PairCreateResp, PairRedeemReq, PairRedeemResp};

use crate::auth::{hash_plaintext, TokenEntry};
use crate::pair::RedeemErr;
use crate::serve::AppState;

pub fn router() -> Router<AppState> {
    Router::new()
        .route("/pair", post(pair_create))
        .route("/pair/redeem", post(pair_redeem))
        .route("/pair/list", get(pair_list))
}

fn require_bearer(state: &AppState, headers: &HeaderMap) -> Result<String, (StatusCode, axum::Json<serde_json::Value>)> {
    let h = headers.get(axum::http::header::AUTHORIZATION)
        .and_then(|v| v.to_str().ok())
        .unwrap_or("");
    let token = h.strip_prefix("Bearer ").unwrap_or("");
    match state.token_store.verify(token) {
        Some(name) => Ok(name),
        None => Err((StatusCode::UNAUTHORIZED, axum::Json(json!({"error": "unauthorized"})))),
    }
}

async fn pair_create(
    State(state): State<AppState>,
    headers: HeaderMap,
    Json(req): Json<PairCreateReq>,
) -> impl IntoResponse {
    if let Err(e) = require_bearer(&state, &headers) { return e.into_response(); }
    let ttl_secs = req.expires_in_sec.clamp(60, 900);
    let ttl = Duration::from_secs(ttl_secs as u64);
    let (code, expires_at) = match state.pair_registry.create(req.name, ttl) {
        Ok(v) => v,
        Err(e) => return (StatusCode::INTERNAL_SERVER_ERROR,
            axum::Json(json!({"error": "registry_full", "detail": e.to_string()}))).into_response(),
    };
    let invite_url = format!("{}/invite/{}",
        state.public_url.trim_end_matches('/'), code);
    let resp = PairCreateResp { code, expires_at, invite_url };
    (StatusCode::OK, axum::Json(resp)).into_response()
}

async fn pair_redeem(
    State(state): State<AppState>,
    ConnectInfo(addr): ConnectInfo<SocketAddr>,
    Json(req): Json<PairRedeemReq>,
) -> impl IntoResponse {
    match state.pair_registry.redeem(&req.code, addr.ip()) {
        Ok(name) => {
            let plaintext = format!("tnl_{}",
                crate::commands::token::random_token_suffix(26));
            let hash = match hash_plaintext(&plaintext) {
                Ok(h) => h,
                Err(_) => return (StatusCode::INTERNAL_SERVER_ERROR,
                    axum::Json(json!({"error": "internal"}))).into_response(),
            };
            if state.token_store.append(TokenEntry { name: name.clone(), hash }).is_err() {
                return (StatusCode::INTERNAL_SERVER_ERROR,
                    axum::Json(json!({"error": "internal"}))).into_response();
            }
            let resp = PairRedeemResp { token: plaintext, endpoint: state.public_url.clone(), name };
            (StatusCode::OK, axum::Json(resp)).into_response()
        }
        Err(RedeemErr::NotFound) =>
            (StatusCode::NOT_FOUND, axum::Json(json!({"error": "pair_not_found"}))).into_response(),
        Err(RedeemErr::Expired) =>
            (StatusCode::GONE, axum::Json(json!({"error": "pair_expired"}))).into_response(),
        Err(RedeemErr::TooManyAttempts) =>
            (StatusCode::TOO_MANY_REQUESTS, axum::Json(json!({"error": "pair_too_many_attempts"}))).into_response(),
        Err(RedeemErr::RateLimited { retry_after_sec }) =>
            (StatusCode::TOO_MANY_REQUESTS, axum::Json(json!({
                "error": "rate_limited", "retry_after": retry_after_sec
            }))).into_response(),
    }
}

async fn pair_list(State(state): State<AppState>, headers: HeaderMap) -> impl IntoResponse {
    if let Err(e) = require_bearer(&state, &headers) { return e.into_response(); }
    let items: Vec<serde_json::Value> = state.pair_registry.list().into_iter()
        .map(|(code, name, expires_at)| json!({"code": code, "name": name, "expires_at": expires_at}))
        .collect();
    (StatusCode::OK, axum::Json(items)).into_response()
}
```

```rust
use anyhow::{Context, Result};
use url::Url;

#[derive(Debug)]
pub struct Invite { pub endpoint: String, pub code: String }

pub fn parse(invite_url: &str) -> Result<Invite> {
    let u = Url::parse(invite_url).context("parse invite URL")?;
    let scheme = u.scheme();
    if scheme != "http" && scheme != "https" {
        anyhow::bail!("invite URL must be http or https, got {scheme}");
    }
    let mut segs = u.path().trim_start_matches('/').splitn(2, '/');
    let kind = segs.next().unwrap_or("");
    let code = segs.next().unwrap_or("");
    if kind != "invite" || code.is_empty() {
        anyhow::bail!("invite URL path must be /invite/<code>");
    }
    let mut e = u.clone();
    e.set_path(""); e.set_query(None); e.set_fragment(None);
    Ok(Invite {
        endpoint: e.as_str().trim_end_matches('/').to_string(),
        code: code.to_string(),
    })
}
```

```rust
impl Registry {
    pub fn gc_disconnected(&self, grace: Duration) {
        let now = Instant::now();
        self.inner.retain(|_, t| match &t.state {
            SessionState::Active { .. } => true,
            SessionState::Disconnected { since } => now.duration_since(*since) <= grace,
        });
    }

    pub fn mark_disconnected(&self, tunnel_id: ulid::Ulid) {
        if let Some(mut t) = self.inner.get_mut(&tunnel_id) {
            t.state = SessionState::Disconnected { since: Instant::now() };
        }
    }

    pub fn try_reattach(
        &self,
        tunnel_id: ulid::Ulid,
        subdomain: &str,
        owner_token: &str,
        new_handle: Arc<Mutex<dyn tnl_protocol::Session>>,
    ) -> Result<(), &'static str> {
        let mut t = self.inner.get_mut(&tunnel_id).ok_or("not_found")?;
        if t.subdomain != subdomain { return Err("subdomain_mismatch"); }
        if t.owner_token != owner_token { return Err("not_owner"); }
        if matches!(t.state, SessionState::Active { .. }) { return Err("already_active"); }
        t.state = SessionState::Active { handle: new_handle };
        Ok(())
    }
}
```

`inner` here is whatever DashMap your `Registry` uses keyed by `Ulid` (or by subdomain — adapt).

- [ ] **Step 2: Implement `reconnect::run`**

Create `crates/tnl/src/reconnect.rs`:

```rust
use std::time::Duration;

#[derive(Default)]
pub struct Hooks {
    pub cancel_first_session: Option<tokio::sync::oneshot::Receiver<()>>,
}

pub async fn run(
    endpoint: &str,
    token: &str,
    subdomain: Option<&str>,
    local_port: u16,
    mut hooks: Hooks,
) -> anyhow::Result<()> {
    let mut backoff_ms: u64 = 1000;
    let mut last_tunnel: Option<(ulid::Ulid, String)> = None;

    loop {
        let result = if let Some((tid, sub)) = &last_tunnel {
            crate::client::connect_and_reattach(endpoint, token, *tid, sub).await
        } else if let Some(s) = subdomain {
            crate::client::connect_and_create(endpoint, token, s).await
        } else {
            crate::client::connect_and_create_random(endpoint, token).await
        };

        let session = match result {
            Ok(s) => s,
            Err(e) => {
                let recoverable = is_recoverable(&e);
                if !recoverable { return Err(e); }
                eprintln!("warning: connection lost ({e:#}); reconnecting in {}s", backoff_ms / 1000);
                tokio::time::sleep(Duration::from_millis(backoff_ms)).await;
                backoff_ms = (backoff_ms * 2).min(30_000);
                continue;
            }
        };

        // First successful connect: reset backoff and remember the tunnel.
        backoff_ms = 1000;
        last_tunnel = Some((session.tunnel_id, session.subdomain.clone()));

        let cancel = hooks.cancel_first_session.take();
        let accept = tokio::spawn(crate::client::run_accept_loop(session.session, local_port));

        // Wait for either the accept loop to die OR the test hook to fire.
        tokio::select! {
            r = accept => {
                if let Err(e) = r {
                    eprintln!("warning: accept loop ended: {e:#}");
                }
            }
            _ = async {
                if let Some(rx) = cancel { let _ = rx.await; }
                else { std::future::pending::<()>().await; }
            } => {
                eprintln!("[test hook] forced disconnect");
            }
        }
        // Loop reconnects with last_tunnel set, attempting reattach.
    }
}

fn is_recoverable(e: &anyhow::Error) -> bool {
    // Auth failures are not recoverable. Everything else (network/proto) is.
    let s = format!("{e:#}").to_lowercase();
    !(s.contains("unauthor") || s.contains("token rejected"))
}
```

```rust
    let owner = match require_bearer(&state, &headers) { Ok(n) => n, Err(e) => return e.into_response() };
    let all = params.get("all").map(|v| v == "true").unwrap_or(false);
    let infos = state.registry.snapshot_infos();
    let filtered: Vec<_> = if all { infos } else {
        infos.into_iter().filter(|i| i.owner_token == owner).collect()
    };
    (StatusCode::OK, axum::Json(filtered)).into_response()
}

async fn tunnels_delete(
    State(state): State<AppState>,
    headers: HeaderMap,
    Path(subdomain): Path<String>,
) -> impl IntoResponse {
    let owner = match require_bearer(&state, &headers) { Ok(n) => n, Err(e) => return e.into_response() };
    match state.registry.close_by_subdomain(&subdomain, &owner) {
        Ok(()) => (StatusCode::NO_CONTENT, axum::Json(json!({}))).into_response(),
        Err("not_found") => (StatusCode::NOT_FOUND, axum::Json(json!({"error": "not_found"}))).into_response(),
        Err("not_owner") => (StatusCode::FORBIDDEN, axum::Json(json!({"error": "not_owner"}))).into_response(),
        Err(e) => (StatusCode::INTERNAL_SERVER_ERROR, axum::Json(json!({"error": e}))).into_response(),
    }
}
```

```rust
use std::time::{Duration, SystemTime};

#[derive(Debug, Clone, serde::Serialize)]
pub struct LogLine {
    pub timestamp:   SystemTime,
    pub method:      String,
    pub path:        String,
    pub status:      Option<u16>,
    pub duration_ms: u64,
    pub bytes_in:    u64,
    pub bytes_out:   u64,
    pub remote_ip:   Option<String>,
}

#[derive(Clone, Copy, Debug)]
pub enum Verbosity { Quiet, Default, Verbose, VeryVerbose }

#[derive(Clone, Copy, Debug)]
pub enum Format { Text, Json }

pub struct Inspector {
    rx: tokio::sync::mpsc::Receiver<LogLine>,
    verbosity: Verbosity,
    format: Format,
    use_color: bool,
}

impl Inspector {
    pub fn new(rx: tokio::sync::mpsc::Receiver<LogLine>, verbosity: Verbosity, format: Format) -> Self {
        let use_color = std::io::IsTerminal::is_terminal(&std::io::stdout())
            && std::env::var_os("NO_COLOR").is_none();
        Self { rx, verbosity, format, use_color }
    }

    pub async fn run(mut self) {
        while let Some(line) = self.rx.recv().await {
            if matches!(self.verbosity, Verbosity::Quiet) { continue; }
            match self.format {
                Format::Json => {
                    if let Ok(s) = serde_json::to_string(&line) { println!("{s}"); }
                }
                Format::Text => self.print_text(&line),
            }
        }
    }

    fn print_text(&self, l: &LogLine) {
        let t = format!("{}", chrono_like_hms(l.timestamp));
        let status_str = l.status.map(|s| s.to_string()).unwrap_or_else(|| "?".into());
        let status_col = if self.use_color { colorise_status(l.status) } else { status_str.clone() };
        println!("{}  {:<6} {:<24} {}  {:>6}ms  {:>8}  {:>8}",
            t, l.method, truncate(&l.path, 24), status_col,
            l.duration_ms,
            fmt_bytes(l.bytes_in),
            fmt_bytes(l.bytes_out));
    }
}

fn chrono_like_hms(t: SystemTime) -> String {
    let d = t.duration_since(SystemTime::UNIX_EPOCH).unwrap_or_default();
    let secs = d.as_secs() % 86_400;
    let ms = d.subsec_millis();
    let (h, rem) = (secs / 3600, secs % 3600);
    let (m, s) = (rem / 60, rem % 60);
    format!("{:02}:{:02}:{:02}.{:03}", h, m, s, ms)
}

fn truncate(s: &str, n: usize) -> String {
    if s.len() <= n { s.to_string() } else { format!("{}…", &s[..n - 1]) }
}

fn fmt_bytes(n: u64) -> String {
    if n < 1024 { format!("{n}B") }
    else if n < 1024 * 1024 { format!("{:.1}KB", n as f64 / 1024.0) }
    else { format!("{:.1}MB", n as f64 / (1024.0 * 1024.0)) }
}

fn colorise_status(s: Option<u16>) -> String {
    use nu_ansi_term::Color::*;
    let raw = s.map(|s| s.to_string()).unwrap_or_else(|| "?".into());
    match s.map(|s| s / 100) {
        Some(2) => Green.paint(raw).to_string(),
        Some(3) => Blue.paint(raw).to_string(),
        Some(4) => Yellow.paint(raw).to_string(),
        Some(5) => Red.paint(raw).to_string(),
        _ => raw,
    }
}
```

```rust
use std::sync::atomic::{AtomicU64, Ordering};
use std::time::Instant;
use bytes::BytesMut;
use tokio::sync::mpsc;
use crate::inspector::LogLine;

pub async fn forward_with_inspection(
    yamux_stream: impl tokio::io::AsyncRead + tokio::io::AsyncWrite + Unpin + Send + 'static,
    local: tokio::net::TcpStream,
    log_tx: mpsc::Sender<LogLine>,
) -> anyhow::Result<()> {
    use tokio::io::AsyncReadExt;
    use tokio::io::AsyncWriteExt;
    let started = Instant::now();
    let (mut yr, mut yw) = tokio::io::split(yamux_stream);
    let (mut tr, mut tw) = tokio::io::split(local);

    let req_peek = std::sync::Arc::new(parking_lot::Mutex::new(BytesMut::with_capacity(4096)));
    let resp_peek = std::sync::Arc::new(parking_lot::Mutex::new(BytesMut::with_capacity(512)));
    let bytes_in = std::sync::Arc::new(AtomicU64::new(0));
    let bytes_out = std::sync::Arc::new(AtomicU64::new(0));

    let req_peek2 = req_peek.clone();
    let bin = bytes_in.clone();
    let fwd_req = async move {
        let mut buf = [0u8; 8192];
        loop {
            let n = yr.read(&mut buf).await?;
            if n == 0 { break; }
            {
                let mut p = req_peek2.lock();
                if p.len() < 4096 {
                    let take = (4096 - p.len()).min(n);
                    p.extend_from_slice(&buf[..take]);
                }
            }
            tw.write_all(&buf[..n]).await?;
            bin.fetch_add(n as u64, Ordering::Relaxed);
        }
        tw.shutdown().await?;
        Ok::<_, anyhow::Error>(())
    };

    let resp_peek2 = resp_peek.clone();
    let bout = bytes_out.clone();
    let fwd_resp = async move {
        let mut buf = [0u8; 8192];
        loop {
            let n = tr.read(&mut buf).await?;
            if n == 0 { break; }
            {
                let mut p = resp_peek2.lock();
                if p.len() < 512 {
                    let take = (512 - p.len()).min(n);
                    p.extend_from_slice(&buf[..take]);
                }
            }
            yw.write_all(&buf[..n]).await?;
            bout.fetch_add(n as u64, Ordering::Relaxed);
        }
        yw.shutdown().await?;
        Ok::<_, anyhow::Error>(())
    };

    let (_a, _b) = tokio::try_join!(fwd_req, fwd_resp)?;

    let head = req_peek.lock().to_vec();
    let resp = resp_peek.lock().to_vec();
    let (method, path) = peek::parse_request_line(&head)
        .unwrap_or_else(|| ("?".into(), "?".into()));
    let status = peek::parse_response_status(&resp);

    let _ = log_tx.send(LogLine {
        timestamp: std::time::SystemTime::now(),
        method, path, status,
        duration_ms: started.elapsed().as_millis() as u64,
        bytes_in: bytes_in.load(Ordering::Relaxed),
        bytes_out: bytes_out.load(Ordering::Relaxed),
        remote_ip: None,
    }).await;
    Ok(())
}
```

```rust
// crates/tnl/src/exit.rs
pub const EX_OK: i32 = 0;
pub const EX_GENERIC: i32 = 1;
pub const EX_USAGE: i32 = 2;
pub const EX_NOT_AUTH: i32 = 64;
pub const EX_TOKEN_REJECTED: i32 = 65;
pub const EX_SUBDOMAIN_CONFLICT: i32 = 66;
pub const EX_LOCAL_UNREACHABLE: i32 = 67;
pub const EX_SERVER_UNREACHABLE: i32 = 68;
pub const EX_SERVER_ERROR: i32 = 70;
pub const EX_PAIRING_FAILURE: i32 = 75;

/// Classify an anyhow::Error chain into a sysexits-style code.
pub fn classify(e: &anyhow::Error) -> i32 {
    let s = format!("{e:#}").to_lowercase();
    if s.contains("not authenticated") || s.contains("no config") { return EX_NOT_AUTH; }
    if s.contains("token rejected") || s.contains("unauthorized") { return EX_TOKEN_REJECTED; }
    if s.contains("subdomain") && s.contains("conflict") { return EX_SUBDOMAIN_CONFLICT; }
    if s.contains("connection refused") && s.contains("127.0.0.1") { return EX_LOCAL_UNREACHABLE; }
    if s.contains("dns") || s.contains("tls") || s.contains("connect to") { return EX_SERVER_UNREACHABLE; }
    if s.contains("/healthz returned 5") || s.contains("server error") { return EX_SERVER_ERROR; }
    if s.contains("pair_") { return EX_PAIRING_FAILURE; }
    EX_GENERIC
}
```

## Phase 15 — `scripts/install.sh` (client installer)

### Task 35: Implement `scripts/install.sh`

**Files:**
- Create: `scripts/install.sh`

- [ ] **Step 1: Write the script**

POSIX `sh`, ~300 LOC. Reference algorithm from spec §4. Key sections:

```sh
#!/bin/sh
set -e

TNL_VERSION="${TNL_VERSION:-latest}"
TNL_INSTALL_DIR="${TNL_INSTALL_DIR:-$HOME/.local/bin}"
TNL_REPO="${TNL_REPO:-atassis/tnl}"

die() { echo "error: $*" >&2; exit 1; }

# 1. Detect OS / arch
os="$(uname -s | tr A-Z a-z)"
case "$os" in linux|darwin) ;; *) die "unsupported os: $os" ;; esac
arch="$(uname -m)"
case "$arch" in x86_64|amd64) arch=x86_64 ;;
                aarch64|arm64) arch=aarch64 ;;
                *) die "unsupported arch: $arch" ;; esac

# 2. Resolve version
if [ "$TNL_VERSION" = "latest" ]; then
    TNL_VERSION="$(curl -fsSL "https://api.github.com/repos/$TNL_REPO/releases/latest" \
        | sed -n 's/.*"tag_name": *"\([^"]*\)".*/\1/p')"
    [ -n "$TNL_VERSION" ] || die "failed to resolve latest version"
fi

# 3. Download + verify
tmp="$(mktemp -d)"
trap 'rm -rf "$tmp"' EXIT
asset="tnl-${TNL_VERSION}-${arch}-${os}.tar.gz"
url="https://github.com/$TNL_REPO/releases/download/$TNL_VERSION/$asset"
echo "Downloading $asset ..."
curl -fL "$url" -o "$tmp/tnl.tar.gz"
curl -fL "${url}.sha256" -o "$tmp/tnl.tar.gz.sha256"
( cd "$tmp" && {
    if command -v sha256sum >/dev/null 2>&1; then
        sha256sum -c tnl.tar.gz.sha256
    else
        shasum -a 256 -c tnl.tar.gz.sha256
    fi
}) || die "checksum mismatch"

# 4. Install
mkdir -p "$TNL_INSTALL_DIR"
tar -xzf "$tmp/tnl.tar.gz" -C "$tmp"
mv "$tmp/tnl" "$TNL_INSTALL_DIR/tnl"
chmod +x "$TNL_INSTALL_DIR/tnl"

# 5. PATH check
case ":$PATH:" in *":$TNL_INSTALL_DIR:"*) ;;
    *) echo
       echo "$TNL_INSTALL_DIR is not in your PATH. Add to your shell rc:"
       echo "    export PATH=\"$TNL_INSTALL_DIR:\$PATH\""
       echo ;;
esac

echo "Installed: $("$TNL_INSTALL_DIR/tnl" version 2>/dev/null || echo "$TNL_INSTALL_DIR/tnl")"
```

### Task 38: tnld-install.sh

**Files:**
- Create: `scripts/tnld-install.sh`

- [ ] **Step 1: Write the script**

POSIX `sh`. Algorithm from spec §4.

```sh
#!/bin/sh
set -e
TNL_VERSION="${TNL_VERSION:-latest}"
GHCR_IMAGE="${GHCR_IMAGE:-ghcr.io/atassis/tnld}"
NON_INTERACTIVE="${NON_INTERACTIVE:-0}"

die() { echo "error: $*" >&2; exit 1; }

# Preflight
[ "$(id -u)" -eq 0 ] || die "must run as root (or via sudo)"
command -v docker >/dev/null || die "docker not installed; see https://docs.docker.com/engine/install/"
docker compose version >/dev/null 2>&1 || die "docker compose plugin missing"

# Resolve version
if [ "$TNL_VERSION" = "latest" ]; then
    TNL_VERSION="$(curl -fsSL "https://api.github.com/repos/atassis/tnl/releases/latest" \
        | sed -n 's/.*"tag_name": *"\([^"]*\)".*/\1/p')"
fi

# Pull image
docker pull "${GHCR_IMAGE}:${TNL_VERSION#v}"
IMAGE_TAG="${GHCR_IMAGE}:${TNL_VERSION#v}"

# Config locations
mkdir -p /etc/tnld /opt/tnld
chown root:root /etc/tnld
chmod 755 /etc/tnld

# Interactive config OR env-driven
if [ "$NON_INTERACTIVE" = 1 ]; then
    [ -n "$PUBLIC_URL" ] || die "NON_INTERACTIVE=1 requires PUBLIC_URL"
    [ -n "$HOSTNAME_ROOT" ] || die "NON_INTERACTIVE=1 requires HOSTNAME_ROOT"
    cat > /etc/tnld/config.toml <<EOF
listen        = "127.0.0.1:7777"
public_url    = "$PUBLIC_URL"
hostname_root = "$HOSTNAME_ROOT"
tokens_file   = "/etc/tnld/tokens.toml"
EOF
    chmod 644 /etc/tnld/config.toml
    touch /etc/tnld/tokens.toml
    chmod 644 /etc/tnld/tokens.toml
else
    docker run --rm -it -v /etc/tnld:/etc/tnld "$IMAGE_TAG" init
fi

# Compose file
cat > /opt/tnld/compose.yaml <<EOF
services:
  tnld:
    image: ${IMAGE_TAG}
    container_name: tnld
    restart: always
    network_mode: host
    user: "tnld"
    read_only: true
    tmpfs:
      - /tmp:size=16M
    environment:
      RUST_LOG: "info,tnld=debug"
    volumes:
      - /etc/tnld:/etc/tnld:rw
    healthcheck:
      test: ["CMD", "tnld", "healthcheck", "--config", "/etc/tnld/config.toml"]
      interval: 10s
      timeout: 3s
      start_period: 5s
EOF

docker compose -f /opt/tnld/compose.yaml up -d

# Wait for healthy
echo "Waiting for tnld to become healthy..."
for _ in $(seq 1 30); do
    h="$(docker inspect --format='{{.State.Health.Status}}' tnld 2>/dev/null || echo none)"
    [ "$h" = "healthy" ] && break
    sleep 1
done
[ "$h" = "healthy" ] || die "tnld did not become healthy within 30s"

# First admin invite
if [ "$NON_INTERACTIVE" != 1 ]; then
    echo "Creating first admin token..."
    docker exec tnld tnld pair add admin \
        --endpoint http://127.0.0.1:7777 \
        --admin-token DUMMY_FOR_BOOTSTRAP 2>/dev/null || true
    # Bootstrap problem: there's no admin token yet to POST /pair.
    # Solution: tnld init wizard offers to create the first token via
    # direct tokens.toml write; pair add is for subsequent users.
fi

# Quick-start checklist
cat <<'EOF'

Done. Quick-start checklist:
  [✓] /opt/tnld/compose.yaml — running
  [✓] /etc/tnld/{config,tokens}.toml
  [ ] reverse proxy passes *.<hostname_root> → 127.0.0.1:7777
  [ ] wildcard cert for *.<hostname_root> (DNS-01 challenge)
  [ ] wildcard A record *.<hostname_root> → this host's public IP

See /usr/share/doc/tnld/proxy-examples/ inside the container (or
docs/dns-providers.md in the repo) for per-provider snippets.
EOF
```

```bash
Preflights docker + compose, pulls ghcr image, runs tnld init for
```

### Task 48: `.github/workflows/release.yml`

Trigger on `v*.*.*` tag push. Build matrix per spec §7.3. Use:

Pseudo-structure:

```yaml
name: release
on:
  push: { tags: ['v*'] }

jobs:
  build:
    strategy:
      matrix:
        include:
          - target: x86_64-unknown-linux-musl
            runner: ubuntu-latest
          - target: aarch64-unknown-linux-musl
            runner: ubuntu-latest
          - target: x86_64-apple-darwin
            runner: macos-13
          - target: aarch64-apple-darwin
            runner: macos-14
    runs-on: ${{ matrix.runner }}
    steps:
      - uses: actions/checkout@v4
      - uses: houseabsolute/actions-rust-cross@v1
        with:
          target: ${{ matrix.target }}
          args: --release --workspace
      - name: Bundle tarballs
        run: |
          mkdir -p out
          for bin in tnl tnld; do
              dir="${bin}-${{ github.ref_name }}-${{ matrix.target }}"
              mkdir -p "$dir/man/man1"
              cp target/${{ matrix.target }}/release/$bin "$dir/"
              cp target/${{ matrix.target }}/release/build/${bin}-*/out/man/${bin}.1 "$dir/man/man1/" || true
              cp LICENSE-MIT LICENSE-APACHE CHANGELOG.md "$dir/"
              tar czf "out/${dir}.tar.gz" "$dir"
              ( cd out && sha256sum "${dir}.tar.gz" >> SHA256SUMS )
          done
      - uses: actions/upload-artifact@v4
        with: { name: artifacts-${{ matrix.target }}, path: out/ }

  release:
    needs: build
    runs-on: ubuntu-latest
    steps:
      - uses: actions/download-artifact@v4
        with: { path: dist/ }
      - run: |
          mkdir -p flat
          find dist -type f -exec cp {} flat/ \;
      - uses: softprops/action-gh-release@v2
        with:
          files: flat/*
          body_path: .github/release-body.md  # parsed by a separate step
```
