# tnl v0.1.0-alpha Implementation Plan

> Historical planning artifact — captures the plan as written on the date in the filename; the shipped code is the source of truth.

**Goal:** Working reverse-tunnel locally. `tnl http 3000 foo` against in-process `tnld` (no Docker, no remote deploy) successfully forwards an HTTP request with `Host: foo.t.example.com` to a local backend on `127.0.0.1:3000` and returns its response. End-to-end test passes.

**Architecture:** Rust Cargo workspace with three crates: `tnl-protocol` (shared wire types + transport traits), `tnld` (broker daemon), `tnl` (client CLI). Transport in v0.1.0-alpha = yamux multiplexed over WebSocket (in-memory `tokio::io::duplex` in unit tests; WSS over TCP in the integration test). `tnld` holds in-memory registry of tunnels and sessions; on incoming HTTP request to `foo.t.example.com` it looks up the session, opens a yamux substream to the CLI, and bidi-copies raw HTTP/1.1 bytes between the upstream connection and the substream. The CLI accepts substreams and pipes them to `127.0.0.1:<port>`.

**Tech Stack** (versions reflect crates.io heads as of 2026-05-27; pre-release tracks avoided):
- Rust 1.94 (pinned via `rust-toolchain.toml`)
- `tokio` 1.x — async runtime
- `axum` 0.8 + `tower-http` 0.6 — HTTP server, middleware
- `hyper` 1.x — HTTP client/server primitives
- `tokio-tungstenite` 0.29 — WebSocket
- `ws_stream_tungstenite` 0.15 — `AsyncRead+AsyncWrite` adapter over a `WebSocketStream` of binary frames
- `yamux` 0.13 (libp2p) — stream multiplexing
- `reqwest` 0.13 — HTTPS client for REST calls in CLI
- `clap` 4 (derive) — CLI parsing
- `serde` + `serde_json` + `toml` 1.x — config + wire
- `argon2` 0.5 (RustCrypto; latest stable, 0.6 is rc) — token hashing
- `ulid` 1 — tunnel IDs
- `dashmap` 6 (latest stable, 7 is rc) — concurrent maps in registry
- `arc-swap` 1.9 — atomic Arc swapping in `Tunnel.session`
- `tracing` + `tracing-subscriber` — structured logging
- `anyhow` + `thiserror` 2 — error handling
- `nu-ansi-term` — colour formatting (used later; declared now)

**Note on axum 0.8 vs the snippets in this plan:** Most snippets were drafted before bumping axum 0.7 → 0.8. The two breaking changes that matter:
1. Path parameters now use `{name}` syntax, not `:name`. The v0.1.0-alpha routes (`/healthz`, `/whoami`, `/control`, the data-plane fallback) have **no** path params, so this is not a problem in any task.
2. Handler return-type bounds were tightened; `impl IntoResponse` still works as shown. `axum::extract::ws::WebSocketUpgrade::on_upgrade` is unchanged.

If an implementer hits an axum 0.8-specific compile error not anticipated by the snippet, adapt minimally and report under `DONE_WITH_CONCERNS` so the next task can be aware.

**Note on `async-tungstenite` vs `tokio-tungstenite` (substitution applies to Tasks 13 and 18):** Task 7 (commit `dfd361f`) discovered that `ws_stream_tungstenite` 0.15 (the `AsyncRead+AsyncWrite` adapter required to feed yamux) was written against `async-tungstenite`'s `WebSocketStream` types, not `tokio-tungstenite`'s. As a result, the workspace ships `async-tungstenite = { version = "0.29", features = ["tokio-runtime"] }` and removes `tokio-tungstenite` from direct deps.

**For Task 13 (tnld `/control` handler):** the plan's snippet uses `tokio_tungstenite::WebSocketStream::from_raw_socket` for bridging axum's `WebSocket` to a stream that yamux can consume. Substitute `async_tungstenite::WebSocketStream::from_raw_socket` (and `async_tungstenite::tungstenite::protocol::Role`). The bridge structure is otherwise identical.

**For Task 18 (tnl `http` command):** the plan's snippet uses `tokio_tungstenite::connect_async`. Substitute `async_tungstenite::tokio::connect_async` — same async signature, returns `WebSocketStream<TokioAdapter<TcpStream>>` directly usable by `client_session_from_ws` from `tnl_protocol::transport`. Similarly, `tokio_tungstenite::tungstenite::handshake::client::generate_key` → `async_tungstenite::tungstenite::handshake::client::generate_key`.

**Spec deviation note (intentional):** The spec says Caddy uses `reverse_proxy h2c://127.0.0.1:7777`. For v0.1.0-alpha simplicity, we use plain HTTP/1.1 with `header_up Connection close`, so each upstream request is a fresh TCP connection. This eliminates the need for h2c framing on the data plane and lets axum see one request per connection cleanly. Documented in the deploy snippet below. Future revisions may switch to h2c if performance demands.

**What this plan does NOT implement (deferred to v0.1.0-beta):**

- Inspector (live request log in CLI stdout)
- `tnl status` / `tnl stop` subcommands
- Reconnect / reattach window
- IP allow-list / basic auth on tunnels
- `tnld token add/list/revoke` admin subcommands (tokens.toml edited by hand for now; we ship a `tnld hash-password` utility)
- Rate limiting / anti-bruteforce
- Healthcheck subcommand
- Dockerfile / compose / CI
- QUIC transport
- Subdomain auto-generation (subdomain is **required** in v0.1.0-alpha for simplicity)

## Phase 2 — Transport (yamux over WSS)

The Transport layer abstracts how a CLI session talks to the daemon. v0.1.0-alpha ships **one** transport: yamux multiplexed over a WebSocket connection.

Key design constraint: the **daemon must be able to open substreams to the client** (because requests arrive at the daemon first). In yamux this means the **daemon side acts as the yamux Client role**, and the **CLI side acts as the yamux Server role**, regardless of who dialed the underlying connection. This is opposite to the natural intuition — write it down in code comments.

use std::pin::Pin;
use std::time::Duration;

use async_trait::async_trait;
use tokio::io::{AsyncRead, AsyncWrite};

/// A multiplexed substream — must behave as a bidirectional byte channel.
pub trait Stream: AsyncRead + AsyncWrite + Send + Unpin {}
impl<T: AsyncRead + AsyncWrite + Send + Unpin + ?Sized> Stream for T {}

/// A logical session between a `tnl` client and the `tnld` daemon.
/// Provides substream multiplexing. Both ends can both open and accept.
#[async_trait]
pub trait Session: Send + Sync {
    /// Initiate a new outgoing substream toward the peer.
    async fn open_stream(&mut self) -> anyhow::Result<Pin<Box<dyn Stream>>>;

    /// Accept the next incoming substream initiated by the peer.
    /// Returns `None` when the session is closed.
    async fn accept_stream(&mut self) -> anyhow::Result<Option<Pin<Box<dyn Stream>>>>;

    /// Send a ping and await pong; returns the round-trip time.
    async fn ping(&mut self) -> anyhow::Result<Duration>;

    /// Gracefully close the session.
    async fn close(&mut self) -> anyhow::Result<()>;
}

#[cfg(test)]
mod tests {
    use super::*;

    /// Compile-time assertion that the trait is object-safe.
    #[allow(dead_code)]
    fn _is_object_safe(_: Box<dyn Session>) {}
}
```

```rust
use std::collections::VecDeque;
use std::sync::Arc;
use std::task::{Context, Poll};

use futures::stream::StreamExt as _;
use tokio::io::{AsyncReadExt, AsyncWriteExt};
use tokio::sync::Mutex;
use yamux::{Config, Connection, Mode};

/// Yamux-backed multiplexed session over any byte-stream.
///
/// Role mapping: in the tnl/tnld system the **daemon side** is yamux `Mode::Client`
/// (it opens substreams when end-user requests arrive) and the **CLI side** is
/// yamux `Mode::Server` (it accepts substreams). This is intentional and opposite
/// to whoever dialed the underlying connection.
pub struct YamuxSession<T> {
    inner: Arc<Mutex<YamuxInner<T>>>,
}

struct YamuxInner<T> {
    conn: Connection<T>,
    pending_inbound: VecDeque<yamux::Stream>,
}

impl<T> YamuxSession<T>
where
    T: AsyncRead + AsyncWrite + Send + Unpin + 'static,
{
    pub fn new_client(io: T) -> Self {
        let conn = Connection::new(io, Config::default(), Mode::Client);
        Self {
            inner: Arc::new(Mutex::new(YamuxInner {
                conn,
                pending_inbound: VecDeque::new(),
            })),
        }
    }

    pub fn new_server(io: T) -> Self {
        let conn = Connection::new(io, Config::default(), Mode::Server);
        Self {
            inner: Arc::new(Mutex::new(YamuxInner {
                conn,
                pending_inbound: VecDeque::new(),
            })),
        }
    }
}

#[async_trait]
impl<T> Session for YamuxSession<T>
where
    T: AsyncRead + AsyncWrite + Send + Unpin + 'static,
{
    async fn open_stream(&mut self) -> anyhow::Result<Pin<Box<dyn Stream>>> {
        let mut inner = self.inner.lock().await;
        let s = std::future::poll_fn(|cx: &mut Context<'_>| {
            inner.conn.poll_new_outbound(cx)
        })
        .await?;
        Ok(Box::pin(YamuxStreamCompat::new(s)))
    }

    async fn accept_stream(&mut self) -> anyhow::Result<Option<Pin<Box<dyn Stream>>>> {
        let mut inner = self.inner.lock().await;
        if let Some(s) = inner.pending_inbound.pop_front() {
            return Ok(Some(Box::pin(YamuxStreamCompat::new(s))));
        }
        let maybe = std::future::poll_fn(|cx: &mut Context<'_>| {
            inner.conn.poll_next_inbound(cx)
        })
        .await;
        match maybe {
            Some(Ok(s)) => Ok(Some(Box::pin(YamuxStreamCompat::new(s)))),
            Some(Err(e)) => Err(e.into()),
            None => Ok(None),
        }
    }

    async fn ping(&mut self) -> anyhow::Result<Duration> {
        // yamux 0.13 does not expose application-level ping; we open a substream
        // with empty payload and close it immediately as a liveness check.
        let start = std::time::Instant::now();
        let mut s = self.open_stream().await?;
        s.shutdown().await?;
        Ok(start.elapsed())
    }

    async fn close(&mut self) -> anyhow::Result<()> {
        let mut inner = self.inner.lock().await;
        std::future::poll_fn(|cx: &mut Context<'_>| inner.conn.poll_close(cx))
            .await?;
        Ok(())
    }
}

/// Adapter wrapping `yamux::Stream` to expose tokio's `AsyncRead+AsyncWrite`.
struct YamuxStreamCompat {
    inner: tokio_util::compat::Compat<yamux::Stream>,
}

impl YamuxStreamCompat {
    fn new(s: yamux::Stream) -> Self {
        use tokio_util::compat::FuturesAsyncReadCompatExt;
        Self {
            inner: s.compat(),
        }
    }
}

impl AsyncRead for YamuxStreamCompat {
    fn poll_read(
        self: Pin<&mut Self>,
        cx: &mut Context<'_>,
        buf: &mut tokio::io::ReadBuf<'_>,
    ) -> Poll<std::io::Result<()>> {
        Pin::new(&mut self.get_mut().inner).poll_read(cx, buf)
    }
}

impl AsyncWrite for YamuxStreamCompat {
    fn poll_write(
        self: Pin<&mut Self>,
        cx: &mut Context<'_>,
        buf: &[u8],
    ) -> Poll<std::io::Result<usize>> {
        Pin::new(&mut self.get_mut().inner).poll_write(cx, buf)
    }
    fn poll_flush(self: Pin<&mut Self>, cx: &mut Context<'_>) -> Poll<std::io::Result<()>> {
        Pin::new(&mut self.get_mut().inner).poll_flush(cx)
    }
    fn poll_shutdown(self: Pin<&mut Self>, cx: &mut Context<'_>) -> Poll<std::io::Result<()>> {
        Pin::new(&mut self.get_mut().inner).poll_shutdown(cx)
    }
}

use std::path::Path;

use anyhow::Context;
use argon2::password_hash::rand_core::OsRng;
use argon2::password_hash::{PasswordHash, PasswordHasher, PasswordVerifier, SaltString};
use argon2::Argon2;
use serde::{Deserialize, Serialize};

#[derive(Serialize, Deserialize, Debug, Clone)]
pub struct TokenEntry {
    pub name: String,
    pub hash: String,
}

#[derive(Serialize, Deserialize, Debug, Default)]
pub struct TokensFile {
    #[serde(rename = "tokens", default)]
    pub tokens: Vec<TokenEntry>,
}

#[derive(Debug)]
pub struct TokenStore {
    by_hash: HashMap<String, TokenEntry>,
}

impl TokenStore {
    pub fn load(path: &Path) -> anyhow::Result<Self> {
        let text = std::fs::read_to_string(path)
            .with_context(|| format!("read tokens file at {}", path.display()))?;
        let file: TokensFile = toml::from_str(&text).context("parse tokens.toml")?;
        let mut by_hash = HashMap::new();
        for tok in file.tokens {
            by_hash.insert(tok.hash.clone(), tok);
        }
        Ok(Self { by_hash })
    }

    /// Look up which token (if any) matches the given plaintext.
    /// Returns the token name on match, `None` otherwise.
    pub fn verify(&self, plaintext: &str) -> Option<&str> {
        for (hash_str, entry) in &self.by_hash {
            let parsed = match PasswordHash::new(hash_str) {
                Ok(p) => p,
                Err(_) => continue,
            };
            if Argon2::default()
                .verify_password(plaintext.as_bytes(), &parsed)
                .is_ok()
            {
                return Some(&entry.name);
            }
        }
        None
    }

    pub fn is_empty(&self) -> bool {
        self.by_hash.is_empty()
    }
}

/// Hash a plaintext token with argon2id default parameters.
pub fn hash_plaintext(plaintext: &str) -> anyhow::Result<String> {
    let salt = SaltString::generate(&mut OsRng);
    let argon = Argon2::default();
    let hash = argon
        .hash_password(plaintext.as_bytes(), &salt)
        .map_err(|e| anyhow::anyhow!("argon2 hash: {e}"))?;
    Ok(hash.to_string())
}

pub type TunnelId = String;
pub type SessionId = String;

#[derive(Debug)]
pub struct Tunnel {

    pub subdomain: String,
    pub hostname: String,
    pub session_id: SessionId,
    pub created_by: String, // token name
}

#[derive(Debug)]
pub struct SessionState {
    pub id: SessionId,
    pub token_name: String,
}

#[derive(Debug, Default)]
pub struct Registry {
    by_subdomain: DashMap<String, Arc<Tunnel>>,
    by_id: DashMap<TunnelId, Arc<Tunnel>>,
    sessions: DashMap<SessionId, Arc<SessionState>>,
    pub hostname_root: String,
}

#[derive(Debug, Error)]
pub enum RegistryError {
    #[error("subdomain '{0}' is already taken")]
    SubdomainTaken(String),
    #[error("invalid subdomain '{0}'")]
    InvalidSubdomain(String),
    #[error("session not found")]
    SessionMissing,
}

impl Registry {
    pub fn new(hostname_root: impl Into<String>) -> Self {
        Self {
            hostname_root: hostname_root.into(),
            ..Default::default()
        }
    }

    pub fn register_session(&self, token_name: impl Into<String>) -> Arc<SessionState> {
        let id = Ulid::new().to_string();
        let s = Arc::new(SessionState {
            id: id.clone(),
            token_name: token_name.into(),
        });
        self.sessions.insert(id, s.clone());
        s
    }

    pub fn drop_session(&self, session_id: &str) {
        self.sessions.remove(session_id);
        let mut to_remove = Vec::new();
        for kv in self.by_id.iter() {
            if kv.value().session_id == session_id {
                to_remove.push(kv.key().clone());
            }
        }
        for id in to_remove {
            if let Some((_, t)) = self.by_id.remove(&id) {
                self.by_subdomain.remove(&t.subdomain);
            }
        }
    }

    pub fn create_tunnel(
        &self,
        subdomain: &str,
        session_id: &str,
        token_name: &str,
    ) -> Result<Arc<Tunnel>, RegistryError> {
        if !valid_subdomain(subdomain) {
            return Err(RegistryError::InvalidSubdomain(subdomain.into()));
        }
        if !self.sessions.contains_key(session_id) {
            return Err(RegistryError::SessionMissing);
        }
        if self.by_subdomain.contains_key(subdomain) {
            return Err(RegistryError::SubdomainTaken(subdomain.into()));
        }
        let hostname = format!("{subdomain}.{}", self.hostname_root);
        let tunnel = Arc::new(Tunnel {
            id: Ulid::new().to_string(),
            subdomain: subdomain.into(),
            hostname,
            session_id: session_id.into(),
            created_by: token_name.into(),
        });
        self.by_id.insert(tunnel.id.clone(), tunnel.clone());
        self.by_subdomain
            .insert(tunnel.subdomain.clone(), tunnel.clone());
        Ok(tunnel)
    }

    pub fn find_by_hostname(&self, host: &str) -> Option<Arc<Tunnel>> {
        let subdomain = host.strip_suffix(&format!(".{}", self.hostname_root))?;
        self.by_subdomain.get(subdomain).map(|t| t.clone())
    }
}

pub fn valid_subdomain(s: &str) -> bool {
    let len = s.len();
    if !(1..=31).contains(&len) {
        return false;
    }
    let bytes = s.as_bytes();
    if bytes[0] == b'-' || bytes[len - 1] == b'-' {
        return false;
    }
    s.bytes().all(|c| c.is_ascii_lowercase() || c.is_ascii_digit() || c == b'-')
}

use std::sync::Arc;

use anyhow::Context;
use axum::extract::ws::{WebSocket, WebSocketUpgrade};
use axum::extract::State;
use axum::response::IntoResponse;
use tnl_protocol::transport::{server_session_from_ws, Session};
use tnl_protocol::{ControlMsg, ErrorCode, TunnelCreatedResp};
use tokio::io::{AsyncReadExt, AsyncWriteExt};
use tracing::{debug, error, info, warn};

use crate::registry::Registry;
use crate::serve::{AppState, AuthedToken};

pub async fn handler(
    ws: WebSocketUpgrade,
    State(state): State<AppState>,
    axum::Extension(token): axum::Extension<AuthedToken>,
) -> impl IntoResponse {
    // axum's `WebSocketUpgrade::on_upgrade` returns immediately; we wire the
    // upgraded socket to our yamux-over-WS pipeline.
    ws.on_upgrade(move |socket| run_session(socket, state, token))
}

async fn run_session(socket: WebSocket, state: AppState, token: AuthedToken) {
    let registry = state.registry.clone();
    // axum gives us an `axum::extract::ws::WebSocket`; to feed yamux through
    // `ws_stream_tungstenite` we need a `WebSocketStream<S>` — the easiest path
    // is to switch to `tungstenite`'s socket via the lower-level
    // `axum_extra::ws_socket_to_tungstenite` adapter… in v0.1.0-alpha we vendor
    // the conversion inline.
    let ws_stream = match axum_ws_to_tungstenite(socket).await {
        Ok(ws) => ws,
        Err(e) => {
            error!("ws conversion failed: {e}");
            return;
        }
    };

    let mut session = server_session_from_ws(ws_stream);
    let sess_state = registry.register_session(token.name.clone());
    let sess_id = sess_state.id.clone();
    info!(session_id = %sess_id, token = %token.name, "control session opened");

    // The CLI opens the control substream first; we accept it.
    let ctrl_stream = match session.accept_stream().await {
        Ok(Some(s)) => s,
        Ok(None) => {
            warn!("session closed before control stream");
            registry.drop_session(&sess_id);
            return;
        }
        Err(e) => {
            error!("accept control stream: {e}");
            registry.drop_session(&sess_id);
            return;
        }
    };

    if let Err(e) = control_loop(ctrl_stream, &registry, &sess_id, &token.name).await {
        warn!(session_id = %sess_id, "control loop ended: {e}");
    }

    registry.drop_session(&sess_id);
    info!(session_id = %sess_id, "control session closed");
}

async fn control_loop(
    mut stream: std::pin::Pin<Box<dyn tnl_protocol::Stream>>,
    registry: &Registry,
    session_id: &str,
    token_name: &str,
) -> anyhow::Result<()> {
    loop {
        let mut lenbuf = [0u8; 4];
        let read = stream.read_exact(&mut lenbuf).await;
        match read {
            Ok(_) => {}
            Err(e) if e.kind() == std::io::ErrorKind::UnexpectedEof => return Ok(()),
            Err(e) => return Err(e.into()),
        }
        let n = u32::from_be_bytes(lenbuf) as usize;
        if n > 1024 * 1024 {
            return Err(anyhow::anyhow!("control message too large: {n}"));
        }
        let mut payload = vec![0u8; n];
        stream.read_exact(&mut payload).await?;
        let msg: ControlMsg = serde_json::from_slice(&payload).context("decode ControlMsg")?;
        debug!(?msg, "control msg in");

        let response = match msg {
            ControlMsg::CreateTunnel(req) => {
                match registry.create_tunnel(&req.subdomain, session_id, token_name) {
                    Ok(t) => ControlMsg::TunnelCreated(TunnelCreatedResp {
                        tunnel_id: t.id.clone(),
                        hostname: t.hostname.clone(),
                    }),
                    Err(crate::registry::RegistryError::SubdomainTaken(_)) => ControlMsg::Error {
                        code: ErrorCode::SubdomainTaken,
                        message: format!("subdomain '{}' already in use", req.subdomain),
                    },
                    Err(crate::registry::RegistryError::InvalidSubdomain(_)) => ControlMsg::Error {
                        code: ErrorCode::InvalidSubdomain,
                        message: format!("invalid subdomain '{}'", req.subdomain),
                    },
                    Err(e) => ControlMsg::Error {
                        code: ErrorCode::Internal,
                        message: e.to_string(),
                    },
                }
            }
            ControlMsg::Heartbeat => ControlMsg::HeartbeatAck,
            ControlMsg::Close => return Ok(()),
            other => ControlMsg::Error {
                code: ErrorCode::Internal,
                message: format!("unexpected message: {other:?}"),
            },
        };

        let out = serde_json::to_vec(&response)?;
        let len = (out.len() as u32).to_be_bytes();
        stream.write_all(&len).await?;
        stream.write_all(&out).await?;
        stream.flush().await?;
    }
}

/// Convert axum's `WebSocket` to a `tokio_tungstenite::WebSocketStream<S>`
/// so it can plug into `ws_stream_tungstenite`. We do this by extracting the
/// underlying `Upgraded` and rebuilding a `WebSocketStream` on it.
///
/// axum 0.7's `WebSocket` does not expose the `Upgraded` directly; instead it
/// abstracts the message protocol. For v0.1.0-alpha we drive frames through
/// axum's API by spawning a forwarding task that bridges
/// `axum::extract::ws::Message` ↔ `tungstenite::Message`, then we hand the
/// tungstenite side to `WebSocketStream::from_raw_socket` via a `DuplexStream`.
async fn axum_ws_to_tungstenite(
    axum_ws: WebSocket,
) -> anyhow::Result<tokio_tungstenite::WebSocketStream<tokio::io::DuplexStream>> {
    use futures::{SinkExt, StreamExt};

    let (a, b) = tokio::io::duplex(64 * 1024);
    // The yamux/ws_stream_tungstenite layer above will speak binary frames over
    // `WebSocketStream<a>`. Meanwhile we run a bridge that copies tungstenite
    // frames on `b` ↔ axum frames on `axum_ws`. This is one extra hop but
    // keeps the layering simple in v0.1.0-alpha.
    //
    // The bridge:
    //   - reads tungstenite::Message from inner_ws (on `b`), sends as axum::Message
    //   - reads axum::Message from `axum_ws`, sends as tungstenite::Message
    let inner_ws = tokio_tungstenite::WebSocketStream::from_raw_socket(
        b,
        tokio_tungstenite::tungstenite::protocol::Role::Server,
        None,
    )
    .await;
    let outer_ws = tokio_tungstenite::WebSocketStream::from_raw_socket(
        a,
        tokio_tungstenite::tungstenite::protocol::Role::Client,
        None,
    )
    .await;

    // Spawn the bridge between `axum_ws` (real socket) and `inner_ws` (the b side).
    tokio::spawn(async move {
        let (mut inner_sink, mut inner_stream) = inner_ws.split();
        let (mut axum_sink, mut axum_stream) = axum_ws.split();

        let a_to_b = async {
            while let Some(msg) = axum_stream.next().await {
                let m = match msg {
                    Ok(axum::extract::ws::Message::Binary(b)) => {
                        tokio_tungstenite::tungstenite::Message::Binary(b)
                    }
                    Ok(axum::extract::ws::Message::Text(t)) => {
                        tokio_tungstenite::tungstenite::Message::Text(t)
                    }
                    Ok(axum::extract::ws::Message::Ping(b)) => {
                        tokio_tungstenite::tungstenite::Message::Ping(b)
                    }
                    Ok(axum::extract::ws::Message::Pong(b)) => {
                        tokio_tungstenite::tungstenite::Message::Pong(b)
                    }
                    Ok(axum::extract::ws::Message::Close(_)) | Err(_) => break,
                };
                if inner_sink.send(m).await.is_err() {
                    break;
                }
            }
        };

        let b_to_a = async {
            while let Some(msg) = inner_stream.next().await {
                let m = match msg {
                    Ok(tokio_tungstenite::tungstenite::Message::Binary(b)) => {
                        axum::extract::ws::Message::Binary(b)
                    }
                    Ok(tokio_tungstenite::tungstenite::Message::Text(t)) => {
                        axum::extract::ws::Message::Text(t)
                    }
                    Ok(tokio_tungstenite::tungstenite::Message::Ping(b)) => {
                        axum::extract::ws::Message::Ping(b)
                    }
                    Ok(tokio_tungstenite::tungstenite::Message::Pong(b)) => {
                        axum::extract::ws::Message::Pong(b)
                    }
                    Ok(tokio_tungstenite::tungstenite::Message::Close(_)) | Err(_) => break,
                    Ok(tokio_tungstenite::tungstenite::Message::Frame(_)) => continue,
                };
                if axum_sink.send(m).await.is_err() {
                    break;
                }
            }
        };

        tokio::join!(a_to_b, b_to_a);
    });

    Ok(outer_ws)
}

use tnl_protocol::transport::Session;
use tokio::io::{AsyncReadExt, AsyncWriteExt};
use tracing::{debug, error, warn};

use crate::registry::Registry;
use crate::serve::AppState;

pub async fn handler(State(state): State<AppState>, req: Request) -> Response<Body> {
    let host = req
        .headers()
        .get("Host")
        .and_then(|v| v.to_str().ok())
        .map(|s| s.to_string());
    let Some(host) = host else {
        return text(StatusCode::BAD_REQUEST, "missing Host header");
    };

    // Strip any ":port" suffix.
    let host_no_port = host.split(':').next().unwrap_or(&host).to_string();

    let Some(tunnel) = state.registry.find_by_hostname(&host_no_port) else {
        debug!(%host, "no tunnel registered for host");
        return text(StatusCode::BAD_GATEWAY, "no such tunnel\n");
    };

    let session_arc = tunnel.clone();
    let session_id = &session_arc.session_id;

    // For v0.1.0-alpha simplicity we keep a global map session_id -> Sender<Substream>
    // populated by the control handler. This lookup is left for the real wiring
    // when control.rs publishes session handles into state.registry. See
    // SessionHandles below.
    let Some(handle) = state.session_handles.get(session_id).map(|h| h.clone()) else {
        return text(StatusCode::SERVICE_UNAVAILABLE, "client session not ready\n");
    };

    // Acquire the lock and open a new substream on the yamux session.
    let mut session_guard = handle.lock().await;
    let mut substream = match session_guard.open_stream().await {
        Ok(s) => s,
        Err(e) => {
            error!(?e, "open_stream failed");
            return text(StatusCode::SERVICE_UNAVAILABLE, "could not open substream\n");
        }
    };
    drop(session_guard);

    // Write the full HTTP/1.1 request to the substream, then upgrade the
    // upstream connection and bidi-copy.
    let (parts, body) = req.into_parts();
    let head = serialize_http1_request_head(&parts);
    if let Err(e) = substream.write_all(head.as_bytes()).await {
        warn!(?e, "write head to substream");
        return text(StatusCode::BAD_GATEWAY, "write upstream failed\n");
    }

    // Stream the request body (if any) — for non-chunked simple bodies this is
    // the entire content. We rely on Caddy/clients not sending Transfer-Encoding
    // chunked to v0.1.0-alpha.
    let body_bytes = match axum::body::to_bytes(body, 100 * 1024 * 1024).await {
        Ok(b) => b,
        Err(e) => {
            warn!(?e, "read upstream body");
            return text(StatusCode::BAD_GATEWAY, "read req body failed\n");
        }
    };
    if !body_bytes.is_empty() {
        if let Err(e) = substream.write_all(&body_bytes).await {
            warn!(?e, "write body to substream");
            return text(StatusCode::BAD_GATEWAY, "write body failed\n");
        }
    }

    // Read response from substream and stream it back to the caller.
    let mut resp_buf = Vec::with_capacity(8 * 1024);
    let mut tmp = [0u8; 8 * 1024];
    loop {
        match substream.read(&mut tmp).await {
            Ok(0) => break,
            Ok(n) => resp_buf.extend_from_slice(&tmp[..n]),
            Err(e) => {
                warn!(?e, "read substream resp");
                return text(StatusCode::BAD_GATEWAY, "read resp failed\n");
            }
        }
    }

    Response::builder()
        .status(StatusCode::OK)
        .header("Content-Type", "application/octet-stream")
        .body(Body::from(resp_buf))
        .unwrap()
}

fn serialize_http1_request_head(parts: &axum::http::request::Parts) -> String {
    let mut s = String::with_capacity(256);
    s.push_str(&format!(
        "{} {} {:?}\r\n",
        parts.method,
        parts
            .uri
            .path_and_query()
            .map(|p| p.as_str())
            .unwrap_or("/"),
        parts.version
    ));
    for (k, v) in &parts.headers {
        if let Ok(vs) = v.to_str() {
            s.push_str(&format!("{}: {}\r\n", k, vs));
        }
    }
    s.push_str("\r\n");
    s
}

fn text(code: StatusCode, msg: &str) -> Response<Body> {
    Response::builder()
        .status(code)
        .body(Body::from(msg.to_owned()))
        .unwrap()
}
```

use std::pin::Pin;

use anyhow::Context;
use tnl_protocol::Stream;
use tokio::io::{AsyncRead, AsyncWrite};
use tokio::net::TcpStream;
use tracing::{debug, warn};

/// Bidirectionally copy bytes between a yamux substream and a TCP socket to
/// `127.0.0.1:port`. Closes both sides on either EOF.
pub async fn forward(
    mut stream: Pin<Box<dyn Stream>>,
    port: u16,
) -> anyhow::Result<()> {
    let mut tcp = TcpStream::connect(("127.0.0.1", port))
        .await
        .with_context(|| format!("connect 127.0.0.1:{port}"))?;
    tcp.set_nodelay(true)?;
    let (mut sr, mut sw) = tokio::io::split(stream.as_mut());
    let (mut tr, mut tw) = tcp.split();

    let s_to_t = async {
        let n = tokio::io::copy(&mut sr, &mut tw).await?;
        let _ = tw.shutdown().await;
        Ok::<u64, std::io::Error>(n)
    };
    let t_to_s = async {
        let n = tokio::io::copy(&mut tr, &mut sw).await?;
        let _ = sw.shutdown().await;
        Ok::<u64, std::io::Error>(n)
    };
    let (a, b) = tokio::try_join!(s_to_t, t_to_s)?;
    debug!(sent_to_local = a, sent_from_local = b, "stream closed");
    Ok(())
}
```

Create `crates/tnl/src/client.rs`:

```rust
use std::pin::Pin;

use anyhow::{bail, Context};
use tnl_protocol::{ControlMsg, CreateTunnelReq, TunnelCreatedResp};
use tokio::io::{AsyncReadExt, AsyncWriteExt};
use tokio_tungstenite::tungstenite::handshake::client::generate_key;
use tracing::{info, warn};

use crate::forwarder::forward;

pub struct ConnectedSession {
    pub session: Box<dyn tnl_protocol::Session>,
    pub control: Pin<Box<dyn tnl_protocol::Stream>>,
    pub hostname: String,
}

pub async fn connect_and_create(
    endpoint: &str,
    token: &str,
    subdomain: &str,
) -> anyhow::Result<ConnectedSession> {
    // 1. Build WS URL from HTTP endpoint
    let ws_url = endpoint
        .replace("http://", "ws://")
        .replace("https://", "wss://");
    let url = format!("{}/control", ws_url.trim_end_matches('/'));

    let req = http::Request::builder()
        .uri(&url)
        .header("Authorization", format!("Bearer {token}"))
        .header("Host", &url.parse::<http::Uri>().unwrap().authority().unwrap().to_string())
        .header("Upgrade", "websocket")
        .header("Connection", "Upgrade")
        .header("Sec-WebSocket-Key", generate_key())
        .header("Sec-WebSocket-Version", "13")
        .body(())
        .context("build ws upgrade request")?;
    let (ws, _resp) = tokio_tungstenite::connect_async(req)
        .await
        .with_context(|| format!("connect to {url}"))?;
    info!(%url, "connected via WSS");

    // 2. Build CLI-role yamux session
    let mut session = tnl_protocol::transport::client_session_from_ws(ws);

    // 3. Open the control substream
    let mut ctrl: Pin<Box<dyn tnl_protocol::Stream>> = session
        .open_stream()
        .await
        .context("open control substream")?;

    // 4. Send CreateTunnel
    let req = ControlMsg::CreateTunnel(CreateTunnelReq {
        subdomain: subdomain.to_string(),
    });
    let payload = serde_json::to_vec(&req)?;
    let len = (payload.len() as u32).to_be_bytes();
    ctrl.write_all(&len).await?;
    ctrl.write_all(&payload).await?;
    ctrl.flush().await?;

    // 5. Read response
    let mut lenbuf = [0u8; 4];
    ctrl.read_exact(&mut lenbuf).await?;
    let n = u32::from_be_bytes(lenbuf) as usize;
    let mut respbuf = vec![0u8; n];
    ctrl.read_exact(&mut respbuf).await?;
    let resp: ControlMsg = serde_json::from_slice(&respbuf)?;
    let hostname = match resp {
        ControlMsg::TunnelCreated(TunnelCreatedResp { hostname, .. }) => hostname,
        ControlMsg::Error { code, message } => bail!("server error ({:?}): {message}", code),
        other => bail!("unexpected control response: {other:?}"),
    };

    Ok(ConnectedSession {
        session: Box::new(session),
        control: ctrl,
        hostname,
    })
}

pub async fn run_accept_loop(
    mut session: Box<dyn tnl_protocol::Session>,
    port: u16,
) -> anyhow::Result<()> {
    loop {
        match session.accept_stream().await {
            Ok(Some(stream)) => {
                tokio::spawn(async move {
                    if let Err(e) = forward(stream, port).await {
                        warn!(?e, "forward failed");
                    }
                });
            }
            Ok(None) => {
                info!("session closed cleanly");
                return Ok(());
            }
            Err(e) => {
                warn!(?e, "accept_stream error");
                return Err(e);
            }
        }
    }
}
```

    println!("┌─ tnl ─────────────────────────────────────────");

    println!("│ Forward:   127.0.0.1:{port}");

    let accept_handle = tokio::spawn(run_accept_loop(session_box, port));

    // Wait until either accept loop ends or Ctrl-C

        res = accept_handle => {
            res.context("accept loop join")?
        }
        _ = tokio::signal::ctrl_c() => {
            println!("\n✓ stopping tunnel");
            Ok(())
        }
    }
}

If the test fails, common debugging steps:
- Is `data_plane.rs` correctly reading bytes back from the substream? The current implementation reads-until-EOF; the substream must be closed by the CLI side for the daemon to know the response is done. In a real HTTP/1.1 response, the local backend keeps the TCP connection open per keep-alive. The forwarder closes the substream when the local TCP gets EOF; the local backend issues EOF when the response is fully written for Connection: close requests. Therefore Caddy should send `Connection: close` on the upstream hop. In tests we substitute `reqwest` for Caddy; reqwest sends keep-alive by default. Force close by setting:
  ```rust
  let client = reqwest::Client::builder().pool_max_idle_per_host(0).build().unwrap();
  ```
  and add `.header("Connection", "close")`.
- If reqwest's connection-close still doesn't trip EOF, the forwarder needs to detect end-of-response from the parsed `Content-Length` or chunked transfer. v0.1.0-alpha simplifies this: the daemon adds explicit `Connection: close` to the request head it forwards, and the local backend (axum) honors it by closing the TCP after the response is sent.
- Concrete fix in `data_plane.rs` `serialize_http1_request_head` — always inject `Connection: close` and `Host: <local>`:
  ```rust
  s.push_str("Connection: close\r\n");
  s.push_str(&format!("Host: {}\r\n", parts.headers.get("Host").and_then(|v| v.to_str().ok()).unwrap_or("localhost")));
  ```

Apply whichever fix is needed and rerun. The test must pass before commit.

# tnl Deployment Notes (v0.1.0-alpha)

These snippets describe how to wire Caddy in front of `tnld` for tunnel exposure.
v0.1.0-alpha does **not** ship a Dockerfile, compose file, CI, or smoke script —

## What this version requires

- A working wildcard certificate for `*.t.example.com` (DNS-01 with any provider

1. On the host, edit a `tokens.toml`:

   HASH=$(tnld hash-password tnl_DEMOSECRET)

   [[tokens]]
   name = "laptop"
   hash = "${HASH}"
   EOF
   chmod 600 /etc/tnld/tokens.toml

   ```toml
   listen        = "127.0.0.1:7777"

   ```bash
   tnl auth login --endpoint https://tnl-api.t.example.com --token tnl_DEMOSECRET

   tnl http 9999 smoke
   ```
